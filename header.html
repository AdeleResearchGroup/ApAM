Documentation

The APAM documentation covers several topics:

1. Tutorials

Application Abstract Machine (APAM) is a runtime platform to support execution of adaptable applications. It includes the capability to handle dynamic deployment of application's components, as well as opportunistic usage of third-party deployed components.
